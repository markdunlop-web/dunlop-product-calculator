Flexible product calculator system for Dunlop adhesives, levellers, grouts, and other products.

== Description ==

The Dunlop Product Calculator plugin adds customizable calculators to WooCommerce product pages, helping customers determine exactly how much product they need for their projects.

= Features =

* Multiple calculator types (adhesives, levellers, grouts, silicone, waterproofing)
* Easy product configuration through WooCommerce admin
* Responsive design that works on all devices
* Accurate calculations based on industry-standard formulas
* Job list integration ready

= Supported Calculator Types =

* **Powdered Adhesives** - Calculate based on wall/floor application
* **Ready Mixed Adhesives** - Support for wet/dry areas
* **Self-Levelling Compounds** - Depth-based calculations with aggregate options
* **Grout** - Complex tile dimension calculations
* **Silicone Sealant** - Linear coverage calculations
* **Waterproofing** - Area-based kit calculations

== Installation ==

1. Upload the `dunlop-product-calculator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure calculators for individual products in WooCommerce product settings

== Configuration ==

1. Edit any WooCommerce product
2. Scroll to the "Product Data" section
3. Find "Dunlop Calculator Settings"
4. Check "Enable Calculator"
5. Select the calculator type
6. Fill in the relevant fields for your product

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, this plugin extends WooCommerce functionality and requires WooCommerce to be installed and activated.

= Can I use multiple calculator types on the same site? =

Yes, each product can have its own calculator type and configuration.

= Can I customize the calculator appearance? =

Yes, the calculator uses standard CSS classes that can be overridden in your theme.

= Is the plugin mobile responsive? =

Yes, the calculator is fully responsive and works on all device sizes.
